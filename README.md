# spring-boot-mybatis

A demo project for spring boot & druid & mybatis(PageHelper + Mapper) & hot reload application

- Spring boot
- MyBatis
    - Mapper
    - PageHelper
    - MyBatisGenerator build tools
- Druid
- JDBC
- MySQL
- Thymeleaf
